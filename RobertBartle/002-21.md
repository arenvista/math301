# Absolute Value and the Real Line 

The *absolute value* of a real number a, is denoted by $|a|$ is defined by 

$$
\begin{equation}
\left\{ \begin{aligned} 
    a\text{ if } a > 0\\
    0\text{ if } a = 0\\
    -a\text{ if } a < 0
\end{aligned} \right.
\end{equation}
$$
