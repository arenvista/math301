# Absolute Values and The Real Line
## 2.2.1 Absolute Value Defn.
The absolute value of $a \in \mathbb{R}$, denoted $|a|$, is given by:

$$
|a| = 
\begin{cases}
a, a>0 \\
0, a=0 \\
-a, a<0 
\end{cases}
$$

## 2.2.2 Theorem
Consider these values as members of $\mathbb{R}$; The following statements will hold:
* $|ab| = |a||b|$
* $|a|^2 = a^2$
* $c \geq 0 \implies |a| \leq c \iff -c \leq a \leq c$
* $-|a| \leq a \leq |a|$

Abs. values can be used to talk about the *distance* between objects in $\mathbb{R}$
